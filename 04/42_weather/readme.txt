程序中带的ak和sk是方倍的，你可以和我们共用，也可以自己申请一个新的，方法如下：

1. 进入http://lbsyun.baidu.com/apiconsole/key
2. 点击创建应用
应用名称：随便填
应用类型：for server
禁用服务：只选 导航Android SDK
请求校验方式：sn校验方式

点击确认 [如果提示出错，多点几次确认]
3. 确认后返回到api控制台，可看到刚才的应用已创建成功，并且有一个 访问应用(ak)，将ak的值改写到程序中.
4. 点应用配置下的“设置” 在弹出框中可看到 

Security Key：*******显示

点击“显示”，显示一串随机字符，即 sk的值，将其复制后放到程序参数中
